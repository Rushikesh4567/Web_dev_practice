# 🟢 Node.js

## 📌 Introduction
Node.js allows JavaScript to run on the server side. It is used to build scalable backend applications.

---

## 🎯 Learning Objectives
- Understand server-side programming
- Work with modules and file systems

---

## 🧠 Concepts Covered
- Node architecture
- Event loop
- Core modules (fs, http, path)
- NPM (Node Package Manager)

---

## 📂 Practical Work
- Creating HTTP servers
- File handling scripts

---

## 🚀 Key Takeaways
- Node.js is asynchronous and event-driven
- It enables full-stack JavaScript development

---

## 📈 Future Scope
