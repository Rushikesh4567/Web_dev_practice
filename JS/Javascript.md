# ⚡ JavaScript

## 📌 Introduction
JavaScript is the core programming language of the web used to create dynamic and interactive web applications.

---

## 🎯 Learning Objectives
- Understand programming fundamentals
- Manipulate DOM
- Handle asynchronous operations

---

## 🧠 Concepts Covered

### 🔹 Basics
- Variables (`var`, `let`, `const`)
- Data types
- Operators

### 🔹 Control Flow
- if-else
- loops (for, while, for...of, for...in)

### 🔹 Functions
- Function declaration
- Arrow functions
- Callback functions

### 🔹 Data Structures
- Arrays
- Objects

### 🔹 DOM Manipulation
- `getElementById`, `querySelector`
- Event listeners

### 🔹 Events
- Click, input, submit events
- Event bubbling & capturing

### 🔹 Async JavaScript
- setTimeout, setInterval
- Promises
- async/await

### 🔹 Storage
- Local Storage
- Session Storage

---

## 📂 Practical Work
- DOM-based mini projects
- Logic building programs
- Event-driven apps

---

## 🚀 Key Takeaways
- JS adds interactivity to web pages
- Async programming is crucial
- DOM manipulation is core for frontend

---

## 📈 Future Scope
- Advanced JS (closures, prototypes)
