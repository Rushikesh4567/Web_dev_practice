CSS is used to style and layout web pages. This section includes my learning of styling techniques, layout systems, and responsive design.

---

## 🎯 Learning Objectives
- Style HTML elements effectively
- Build responsive layouts
- Use modern layout systems like Flexbox and Grid

### 🔹 Layout Techniques
- Display properties
- Positioning (static, relative, absolute, fixed)

### 🔹 Flexbox
- Flex container & items
- Alignment and spacing

### 🔹 CSS Grid
- Grid layout structure
- Rows, columns, and areas

### 🔹 Responsive Design
- Media queries
- Mobile-first approach

### 🔹 Animations
- Transitions
- Keyframes
